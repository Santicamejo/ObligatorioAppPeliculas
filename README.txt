MovieList – Aplicación Web de Gestión de Películas con APIs e IA

Desarrollo de una aplicación web utilizando Ionic, enfocada en la gestión y evaluación de películas. Implementa consumo de APIs externas, gestión de usuarios (registro e inicio de sesión) y control de errores en las respuestas.

Integra funcionalidades de inteligencia artificial para la evaluación automática de películas a partir de comentarios enviados, permitiendo registrar únicamente aquellas con calificación positiva.

Incluye integración de mapas mediante Leaflet para visualización de la cantidad de usuarios por pais.
